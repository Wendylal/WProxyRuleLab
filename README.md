# WProxyLab 🐿️

面向 Surge / Loon 的配置模板（Configs）与规则文件（Rules）仓库。  
个人维护，侧重 结构化分流**、**场景化策略组 与 **可复用规则集**。

---

## 📦 仓库内容

- Surge/
  - 配置模板（Configs）
  - 规则文件（Rules）
- Loon/
  - 配置模板（Configs）
  - 规则文件（Rules）
- **Docs/**（可选）
  - 说明文档 / 使用指引 / 维护记录

---

## 🚀 使用说明（简要）

1. 选择对应工具目录（`Surge/` 或 `Loon/`）
2. 引用或复制配置模板
3. 按需启用规则文件（建议按场景逐步接入）
4. 将规则引用放在合适顺序（自定义强制规则通常应靠前）

> 💡 建议：首次使用先从最小可用配置开始，再逐步增加场景规则，便于排查与维护。
